# AIHPC-Bootstrap

Training & Development materials for **AI / HPC**, aimed at an audience
ranging from college students to experienced professionals.

## Layout

| Path        | Purpose                                                  |
| ----------- | -------------------------------------------------------- |
| `ai/`       | Context & instructions for AI agents collaborating here  |
| `docs/`     | Markdown documentation                                   |
| `docs/latex/` | LaTeX sources                                          |
| `docs/pdf/` | PDFs generated from `docs/latex`                         |
| `projects/` | Hands-on projects (each in its own numbered subfolder)   |

## Projects

1. [Dev Environment Setup](projects/01-dev-environment-setup/) — build a
   local Ubuntu 26.04 LTS **dev-workstation**, a headless QEMU
   **dev-vm**, and wire up VS Code Remote-SSH against it.

## Ethics & Integrity

covers IEEE Code of Ethics compliance, AI ethics and attribution rules,
and academic / professional integrity expectations for all participants.
